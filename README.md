# Angular

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.2.2.

## Crear un proyecto en Firebase

Luego un database con el nombre "courses"

Agregar los siguientes campos:

```
date
description
language
name
price
teacher
technology
```
